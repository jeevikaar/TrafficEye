# 🚦 TrafficEye – AI-Based Smart Traffic and Route Optimization

TrafficEye is a web-based smart traffic management application that helps users find the most efficient route between two locations. The system analyzes route information, including distance, estimated travel time, traffic conditions, and environmental impact, enabling users to make informed travel decisions. By integrating interactive maps and intelligent route analysis, TrafficEye promotes efficient navigation while encouraging eco-friendly transportation.

---

## 📌 Features

- 🗺️ Interactive map interface using OpenStreetMap
- 📍 Source and destination location selection
- 🛣️ Smart route generation
- 🚦 Traffic condition analysis
- ⏱️ Estimated travel time calculation
- 📏 Distance calculation
- 🌱 Eco Impact analysis
- ⛽ Fuel consumption estimation
- 🌍 CO₂ emission analysis
- 📱 Simple and responsive user interface

## 🚀 How It Works

1. Enter the starting location.
2. Enter the destination.
3. Click **Find Route**.
4. The application displays:
   - Best available route
   - Distance
   - Estimated travel time
   - Traffic status
   - Eco Impact
   - Fuel consumption
   - CO₂ emission details
